IS601 Assignment to learn how to host website using Github pages by using Linus commands, Vi text editor and Git

Linux Commands

  • CD - Change Directory
  • MKDIR - Make Directory
  • LS - List Files

Vi Commands

  • I - Insert mode
  • Esc - Get out of insert mode
  • Shift key + Colon - Command Prompt for vi
  • W - Command to write
  • Q - Command to quit
  • WQ - Write and Quit
